# EL_Tags
Eternal Lands map tags

Don't recall exactly where I got these.  
Possibly from the PEN website, but it's been about 5 years.  

Think it's a gathering of locations from a few sources.  
Back up your tags, if you have any.  
You can always merge those in with a text-editor.  

---

> Windows:  

unzip in `/Documents/Eternal Lands/main`

so everything unfolds into `/Documents/Eternal Lands/main/maps`

---
> Linux:  

unzip in `~/.elc/main`

so ya get everything in `~/.elc/main/maps`
